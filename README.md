# 🌈 Daycare Bot
это Telegram-бот для взаимодействия родителей с детским садом.  
Он позволяет быстро получать информацию о ребёнке, общаться с воспитателем и отправлять важные уведомления.

Программа разработана для курсового проекта "Разработка базы данных для работы групп детского сада"

---

## Возможности

- 🔐 Авторизация по номеру телефона
- 👧 Выбор ребёнка из списка
- 🏫 Просмотр группы ребёнка
- ⏰ Просмотр расписания на сегодня/завтра
- ✉️ Отправка сообщений воспитателю
- 🚪 Уведомление "ребёнок у двери"
- 👕 Запрос на помощь с одеждой
- ❗️ Сообщение о пропуске
- 📱 Контакты воспитателя и директора

---

## Как работает бот

1. Пользователь вводит номер телефона
2. Бот находит родителя в базе данных
3. Показывает список детей
4. После выбора ребёнка открывается меню действий
